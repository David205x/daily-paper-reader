## 摘要
大语言模型能够从自然语言描述生成BPMN过程模型，然而监督微调将输出质量限制在训练数据中的模式。强化学习可以利用外部质量度量超越这一上限，但当质量是多维度时，奖励函数应如何设计尚待探索。我们对基于强化学习的过程模型生成的奖励函数设计进行了系统研究，使用48种配置训练了两个大语言模型家族（Llama 3.1 8B，Qwen 2.5 14B），采用基于组序列策略优化方法，奖励来自一个包含语法、语用和语义质量共38个指标的自动评估框架。三个发现：第一，强化学习显著提升语用和语法质量，同时保持语义保真度，输出变异性降低超过六倍。第二，等权奖励一致优于针对性权重：强调特定维度未能改善该维度，反而可能导致模型陷入低质量模式。第三，设计选择与模型架构以非平凡方式交互：无效惩罚对一个模型至关重要，但对另一个无关紧要；监督微调初始化对一种架构不可或缺，但对另一种适得其反。这些结果表明奖励组合是优化结果的主要决定因素，其影响与是否应用强化学习本身一样大。该发现适用于任何沿着多个自动维度评估质量的结构化生成任务。我们在https://github.com/chlauer99/RL_for_process_modeling发布实现和实验代码。

## Abstract
Large language models (LLMs) can generate BPMN process models from natural-language descriptions, yet supervised fine-tuning (SFT) limits their output quality to the patterns present in the training data. Reinforcement learning (RL) can optimize beyond this ceiling using external quality measures, but how the reward function should be designed when quality is multi-dimensional remains unexplored. We present a systematic investigation of reward function design for RL-based process model generation, training two LLM families (Llama~3.1 8B, Qwen~2.5 14B) under 48 configurations using Group Sequence Policy Optimization with rewards derived from an automated evaluation framework comprising 38 metrics across syntactic, pragmatic, and semantic quality. Three findings emerge. First, RL significantly improves pragmatic and syntactic quality while preserving semantic fidelity, reducing output variability by more than sixfold. Second, equal reward weighting consistently outperforms targeted weighting: emphasizing a specific dimension fails to improve it and can collapse the model into a low-quality mode. Third, design choices interact with model architecture in non-trivial ways: the invalidity penalty is essential for one model but irrelevant for the other, and SFT initialization is indispensable for one architecture but counterproductive for another. These results demonstrate that reward composition is a primary determinant of optimization outcomes, with effects as large as the decision to apply RL itself. The findings generalize to any structured generation task where quality is assessed along multiple automated dimensions. We release our implementation and experimental code at https://github.com/chlauer99/RL_for_process_modeling.

## 6. 主要结论与发现

### 6.1 RQ1：GSPO是否超越SFT？
- **语用质量**：两个模型均显著大幅提升（\(\Delta>0.11, p<0.001\)）。
- **语法质量**：Llama显著提升（\(\Delta=+0.092\)），Qwen小幅显著提升（\(\Delta=+0.032\)）。
- **语义质量**：Llama有小幅显著提升（\(\Delta=+0.030, p=0.018\)），Qwen无显著变化（\(\Delta=-0.018, n.s.\)）。
- **输出变异性**：RL训练后语用质量标准差下降超6倍，分布高度收紧。

### 6.2 RQ2：奖励函数组成如何影响权衡？
- **维度加权**：
  - 等权（\(R_1\)）一致优于所有目标加权（\(R_3\)-\(R_5\)）。目标加权不仅未能提升强调维度，反而导致**策略崩溃**（如Llama的语法加权导致语法得分从0.926降至0.662，语义加权导致语义得分下降）。
  - Qwen上加权虽有部分偏转，但总是向语用简化方向偏移，且语法维度普遍受损。
- **无效惩罚**：
  - 对Llama无显著影响（SFT已解决有效性问题）。
  - 对Qwen至关重要：移除惩罚（\(R_2\)）导致**模板收敛**——模型为所有输入输出几乎相同的简单结构（语法标准差0.002），语用极高但语法和语义大幅下降。惩罚作为隐式多样性正则化。

### 6.3 RQ3：初始化策略的影响
- **Llama**：SFT初始化**必不可少**，untrained基在RL下语法得分仅0.664，远低于SFT基的0.926（\(\Delta=0.262\)，为本研究最大效应之一）。
- **Qwen**：SFT初始化对语法和语用无显著优势，反而在语义质量上显著**低于**untrained基（0.567 vs. 0.601）。表明SFT可能约束了语义探索空间。
- 结论：初始化策略需与模型架构匹配，不能默认SFT→RL为通用最优管道。

### 6.4 其他发现
- 数学等价的\(R_{\text{avg}}\)与\(R_1\)在Llama上产生显著差异（语法\(\Delta=0.059\)，可能归因于实现路径或随机性），在Qwen上无差异，提示较小模型对奖励管道细节更敏感。
- 语法约束解码对表现好的配置作用不大，对弱配置可提升有效性但无法修复质量缺陷。

## 7. 优点

- **系统性**：首次在多维结构化生成任务中系统研究奖励函数设计，覆盖权重、惩罚、初始化、解码等多个维度。
- **评估框架先进**：使用BEF4LLM（38个可验证指标），基于SIQ质量理论，自动化且可解释。

- **可复现性**：公开代码，统一数据集，超参数清晰。

## 8. 不足与局限

### 8.1 实验覆盖
- **评估集仅105个样本**，且单次生成，无法估算配置内方差，对小效应统计力有限。
- **加权比例固定为3:1:1**，未测试更温和（如2:1:1）或极端比例（5:1:1），Llama的完全失败可能不适用于更接近等权的加权。
- **模型数量少**：仅两个家族，且同时存在架构和规模差异，无法分解。
- **解码约束**仅作为测试时因素，未训练时使用，可能低估其潜力。

### 8.2 方法论局限
- **语义质量依赖ground-truth**，可能低估生成有效但结构新颖的模型，造成评估偏差。
- **单epoch GSPO**和**LoRA**虽缓解了过拟合和遗忘，但未显式测试reward overfitting和catastrophic forgetting。
- **GSPO的组相对优势**在候选全差时可能产生噪声信号（如Llama untrained基），但当前设计未提供探索机制防止早熟崩溃。
- 未进行多次运行以评估训练随机性，仅报告单次结果。

### 8.3 适用范围
- 研究聚焦BPMN领域，虽声称可推广至其他结构化生成，但缺乏跨域验证。
- 仅使用GPT-style自回归模型，未测试编码器-解码器架构。
- 奖励函数设计依赖自动评估框架；在无可信自动评估的领域（如创意文本），该方法不直接可用。

### 8.4 其他
- 未讨论计算成本（GPU小时数），影响可复现性评估。
- 未探索自适应权重或动态调整策略。
- 对Qwen的语义质量倒退（SFT基 vs. untrained基）原因分析较浅，缺乏消融实验。
