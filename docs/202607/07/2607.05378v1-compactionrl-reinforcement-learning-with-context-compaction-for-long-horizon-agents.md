## 摘要
长时域智能体语言模型日益受到有限上下文窗口的限制，因为扩展的交互轨迹可能在任务完成之前就超过最大上下文长度。上下文压缩通过总结先前的交互状态并在压缩上下文下继续展开，提供了一种自然的解决方案，但将压缩融入强化学习仍未被充分探索。我们提出CompactionRL，一种使用上下文压缩训练长时域智能体语言模型的强化学习策略。我们的方法通过词元级损失归一化和跨轨迹广义优势估计，联合优化任务执行和摘要生成。该设计使语言模型智能体能够从压缩的长时域轨迹中学习。我们在开放模型上训练CompactionRL，并在智能体编码任务中观察到一致的性能提升。CompactionRL使开放的GLM-4.5-Air模型（106B-A30B）在SWE-bench Verified上达到66.8%的Pass@1分数，在Terminal-Bench 2.0上达到24.5%，分别提升了7.0和3.1个绝对点。基于GLM-4.7-Flash（30B-A3B），CompactionRL将Pass@1分别提升了5.5和6.8个点，在SWE-bench Verified上达到56.0%，在Terminal-Bench 2.0上达到20.2%。因此，CompactionRL被部署在用于训练开放的GLM-5.2模型（750B-A40B）的强化学习流程中。

## Abstract
Long-horizon agentic LLMs are increasingly limited by finite context windows, as extended interaction trajectories can exceed the maximum context length before a task is completed. Context compaction offers a natural solution by summarizing previous interaction states and continuing the rollout under a compressed context, but incorporating compaction into reinforcement learning remains underexplored. We propose CompactionRL, a reinforcement learning strategy to train long-horizon agentic LLMs with context compaction. Our approach jointly optimizes task execution and summary generation with token-level loss normalization and cross-trajectory generalized advantage estimation. This design enables the LLM agents to learn from compacted long-horizon trajectories. We train CompactionRL on top of open models and observe consistent performance gains on agentic coding tasks. CompactionRL enables the open GLM-4.5-Air model (106B-A30B) to achieve Pass@1 scores of 66.8% on SWE-bench Verified and 24.5% on Terminal-Bench 2.0, with absolute gains of 7.0 and 3.1 points, respectively. Built upon GLM-4.7-Flash (30B-A3B), CompactionRL improves Pass@1 by 5.5 and 6.8 points, reaching 56.0% on SWE-bench Verified and 20.2% on Terminal-Bench 2.0, respectively. CompactionRL is thus deployed in the RL pipeline for training the open GLM-5.2 model (750B-A40B).

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **问题**：长时域交互任务（如软件工程、终端问题解决）中，LLM 智能体的交互轨迹会逐渐累积，超出有限的上下文窗口，导致任务中断或性能下降。现有方法多将上下文压缩作为推理时的启发式手段或外部内存操作，但未在强化学习（RL）训练中有效整合。
- **动机**：压缩改变了轨迹结构（分割成多个片段），传统 RL 方法（如 GRPO 的组归一化优势）不再适用。任务成功不仅取决于执行策略，还依赖于压缩摘要的质量。因此需要一种能同时在 RL 训练中优化执行和摘要生成的框架。
- **整体含义**：将上下文压缩融入 RL 训练，可有效扩展智能体的有效训练水平，而无需增加最大工作上下文长度。这为在固定上下文预算下训练长时域 Agent 提供了新范式。

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：提出 CompactionRL，基于 PPO 的 RL 框架，将可训练的上下文压缩整合到 rollout 收集中。当剩余上下文低于阈值时，策略生成一个摘要，然后从摘要和最近少量上下文继续交互。摘要与任务执行动作在同一任务奖励下联合优化。
- **关键技术细节**：
  - **可训练上下文压缩**：在 rollout 中，当 `C - |ht| < T_comp` 时触发压缩，策略生成摘要，之后用摘要 + 最近 k 步重构上下文。摘要由可训练策略采样，参与 RL 优化，不额外设计摘要质量奖励。
  - **优化设计**：
    - **Token 级损失归一化**：由于压缩导致片段数量与长度异构，采用 token 级 PPO 损失（对每个生成 token 计算 ratio 和 clipped surrogate），消除片段计数偏差，使每个 token 等权重。
    - **跨轨迹广义优势估计（Cross-trajectory GAE）**：针对每个片段独立计算局部 GAE 后，根据后续片段 token 数 `N_>s` 对优势进行折扣校正：`Â_{s,i} = (γλ)^{N_>s} * A_loc_{s,i}`，使得奖励的时间折扣与原始完整轨迹一致，避免片段化导致的前期动作过度信用。
  - **整体流程**：每个 rollout 由多个执行/摘要片段组成，共享最终任务奖励。使用 PPO 的 critic 估计价值，避免组大小可变问题。

## 7. 优点
- **创新性**：首次将上下文压缩作为 RL 训练的可优化部分，而非推理时启发式，提出联合优化框架。
- **技术设计合理**：针对压缩导致的片段异构性，设计 token 级损失归一化（消除分配偏置）和跨轨迹 GAE（修正时间信用分配），理论清晰有效。
- **实验证明泛化性**：在两个不同规模的模型（30B、106B）和两个编码基准上均取得一致提升，且消融实验验证了各组件的必要性。
- **实际部署价值**：已用于训练 750B 的 GLM-5.2 模型，说明其实用性。

## 8. 不足与局限
- **训练-测试不匹配**：CompactionRL 在单窗口（禁用压缩）评估下性能不稳定甚至下降，表明它专门针对压缩场景优化，无法直接迁移到非压缩设置。

- **领域偏窄**：实验主要聚焦于编码类长时域任务（SWE、Terminal），未涉及其他如网页交互、具身智能等具有不同观察结构和奖励信号的领域，通用性待验证。
