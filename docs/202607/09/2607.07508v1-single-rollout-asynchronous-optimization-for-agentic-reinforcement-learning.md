## 摘要
强化学习在大型语言模型的后训练中变得越来越重要。之前的LLM强化学习流程大多是同步和批处理交错的，对于长周期智能体任务效率低下。最近，异步强化学习作为一种更高效的替代方案出现，它通过在新展开发步到达时更新模型来实现。然而，现有的异步强化学习系统往往强调吞吐量，而对训练稳定性和任务有效性的探索不足。例如，一个关键挑战是广泛使用的GRPO框架中的分组采样并不自然地适用于异步智能体训练。在本文中，我们提出单展开发步异步优化来解决异步强化学习中的稳定性和离策略挑战。为了减少离策略效应并提高泛化能力，我们使用单展开发步采样替代分组采样，即每个提示使用一个展开发步。我们进一步通过实用的价值模型训练设计改进了这种单展开发步策略。为了提高优化稳定性，我们引入了严格的双侧Token级裁剪策略。SAO能够稳定训练一千步，并在智能体编码和推理基准测试（如SWE-Bench Verified、BeyondAIME和IMOAnswerBench）上一致优于GRPO及其变体。我们还证明，单展开发步强化学习在模拟在线学习环境中特别有效，该环境中模型必须适应不断变化的环境。为此，SAO已成功部署在智能体强化学习流程中，用于训练开源的GLM-5.2模型（750B-A40B）。

## Abstract
Reinforcement learning (RL) is becoming increasingly important for post-training large language models (LLMs). Previous RL pipelines for LLMs were mostly synchronous and batch-interleaved, which is inefficient for long-horizon agentic tasks. Recently, asynchronous RL has emerged as a more efficient alternative by updating the model as rollouts arrive. However, existing asynchronous RL systems often emphasize throughput, while leaving training stability and task effectiveness largely underexplored. For example, a key challenge is that group-wise sampling in the widely-used GRPO framework does not naturally fit asynchronous agentic training. In this paper, we present Single-rollout Asynchronous Optimization (SAO) to address the stability and off-policy challenges in asynchronous RL. To reduce off-policy effects and improve generalization, we replace group-wise sampling with single-rollout sampling, that is, using one rollout per prompt. We further improve this single-rollout strategy with practical value-model training designs. To improve optimization stability, we introduce a strict double-side token-level clipping strategy. SAO is able to train stably for one thousand steps and consistently outperform GRPO and its variants on agentic coding and reasoning benchmarks, such as SWE-Bench Verified, BeyondAIME, and IMOAnswerBench. We also demonstrate that single-rollout RL is particularly effective in a simulated online learning setting, where the model must adapt to changing evolving environments. To this end, SAO is successfully deployed in the agentic RL pipeline for training the open GLM-5.2 model (750B-A40B).

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：大型语言模型（LLM）的后训练中，强化学习（RL）越来越重要。传统的LLM RL流程多为同步和批处理交错的（如PPO、GRPO），需要等待整个批次展开完成后才开始优化。对于长周期智能体任务（如编码、多步推理），展开长度高度可变，导致短轨迹完成后GPU空闲等待慢轨迹（stragglers），资源利用率低。
- **核心问题**：异步RL虽然能通过持续消费到达的展开提升效率，但引入了两个主要挑战：
  1. **训练稳定性**：异步导致“策略滞后”（policy lag），不同轨迹可能由多个版本的旧策略生成，离策略（off-policy）严重，训练不稳定。
  2. **与现有算法不匹配**：广泛使用的GRPO依赖组内采样（group-wise sampling）进行优势估计，需要等待组内所有样本完成才能训练，导致延迟和离策略；且组采样与单轨迹反馈的在线/智能体环境不兼容。
- **研究动机**：提出一种既能保持异步RL效率、又能解决稳定性和离策略问题的新方法，使异步RL在智能体任务中有效且稳定。

## 2. 论文提出的方法论

- **核心思想**：**单展开发步异步优化（SAO）**，以单rollout采样替代GRPO的组采样，结合强化的价值模型训练和严格的token级裁剪，实现稳定有效的异步RL。

- **关键技术细节**：
  1. **直接双面重要性采样（DIS）**：
     - 直接使用展开策略（π_rollout）作为行为代理，与当前策略π_θ计算重要性采样比率 r_t(θ)=π_θ/π_rollout，避免维护多个旧策略（π_θold）的历史。
     - 采用双侧token级裁剪与掩码：将信任区域限制在[1-ε_l, 1+ε_h]内，超出范围的token完全掩码（梯度设为0），防止极端策略偏离导致不稳定。裁剪函数 f(x) 在区间内返回x，否则返回0。
  2. **单rollout采样**：
     - 每个提示只生成一个轨迹，完成后立即送入训练，减少组采样导致的等待和离策略偏差。
     - 为降低单rollout的高方差，设计了更强的价值模型训练策略。
  3. **价值模型训练优化**：
     - **价值更新快于策略（Faster Value Update）**：每对策略进行一次梯度更新，就对价值网络进行K次更新（实验中K=2），使价值估计更快适应策略变化，降低方差。
     - **冻结注意力参数（Frozen-Attention）**：在RL训练中冻结价值模型的注意力模块参数，仅优化混合专家（MoE）层，减少梯度不稳定（观察到全参数训练时注意力层梯度范数大）。
     - **跳过观察的Token级GAE（Skip-Observation GAE）**：针对智能体任务中轨迹包含动作和环境反馈（如“动作-观察-动作”形式），修改贝尔曼目标，将优势估计限制在模型生成的action token之间，跳过observation token，避免环境随机性引入噪声。具体公式：ˆA(a_i,N) = δ + γλ ˆA(a_{i+1,0})，其中δ = r_t + γV(a_{i+1,0}) - V(a_i,N)。
     - **扩展价值预训练数据**：大幅增加价值模型预训练语料规模，解决“冷启动”问题。
