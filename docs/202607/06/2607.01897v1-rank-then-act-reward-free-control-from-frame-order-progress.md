## 摘要
我们提出了“先排序再行动”（Rank-Then-Act, RTA）框架，用于在没有环境奖励的情况下从专家视频演示中学习控制策略。RTA 离线训练一个视觉语言模型（VLM）作为基于进度的序数评分器，通过对打乱的帧序列采用分组相对策略优化（GRPO）目标，迫使模型从视觉语义而非琐碎的时间线索中恢复时间顺序。重要的是，我们不直接将该评分器用作标量奖励模型，而是提出了一种基于相关性的奖励函数用于强化学习：在每个交互窗口，我们计算预测的进度排名与真实时间索引之间的斯皮尔曼等级相关系数，从而得到一个有界、尺度不变的训练信号。这种设计将奖励学习与绝对校准解耦，并使得跨任务和环境的稳定迁移成为可能。我们在离散控制基准（PyBoy: Catrap, Kirby）和连续控制任务（PointMaze, MetaWorld）上评估了 RTA。RTA 始终达到或超越先前的基于视频的奖励学习方法以及基于排名的基线，同时展示了单个预训练进度评分器在跨任务重用中的强大性能。我们的结果表明，基于视频导出的序数信号的相关性结构监督足以用于策略学习，为显式奖励设计提供了一种可扩展的替代方案。

## Abstract
We introduce Rank-Then-Act (RTA), a framework for learning control policies from expert video demonstrations without environment rewards. RTA trains a Vision-Language Model (VLM) offline as a progress-based ordinal scorer, using a Group Relative Policy Optimization (GRPO) objective over shuffled frame sequences, which forces the model to recover temporal ordering from visual semantics rather than trivial time cues. Importantly, instead of using the scorer directly as a scalar reward model, we propose a correlation-based reward function for reinforcement learning: at each interaction window, we compute the Spearman rank correlation between predicted progress rankings and true temporal indices, yielding a bounded, scale-invariant learning signal. This design decouples reward learning from absolute calibration and enables stable transfer across tasks and environments. We evaluate RTA on discrete control benchmarks (PyBoy: Catrap, Kirby) and continuous control tasks (PointMaze, MetaWorld). RTA consistently matches or outperforms prior video-based reward learning methods and rank-based baselines, while demonstrating strong cross-task reuse of a single pretrained progress scorer. Our results suggest that correlation-structured supervision over video-derived ordinal signals is sufficient for policy learning, offering a scalable alternative to explicit reward design.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 利用**帧序的序数监督**（ordinal supervision）替代标量奖励：训练VLM作为进度评分器，并根据预测排名与真实时间戳的**斯皮尔曼等级相关系数（Spearman ρ）** 作为奖励信号，该信号有界∈[-1,1]且尺度不变，从而与绝对校准解耦。

### 关键技术细节（两阶段）

#### 阶段1：Rank – 进度评分器训练
- **输入**：专家视频中抽取的K帧序列，固定第一帧为锚点（anchor），其余帧随机打乱。
- **模型**：LoRA微调的VLM（Qwen2.5-VL-7B），输出每帧的进度排名（越高表示越靠后）。
- **目标**：最大化预测排名与真实时间索引的Spearman ρ。
- **优化**：采用**分组相对策略优化（GRPO）**，对VLM的文本生成进行列表式（listwise）训练，避免“越晚越好”的简单捷径。

#### 阶段2：Act – 在线控制
- **冻结评分器**，用于交互时提供奖励。
- **滑动窗口**：每N步（默认N=15）取最近的m帧（含锚点），对非锚帧进行L次独立随机打乱（默认L=2），分别计算评分器输出的进度排名与时间戳的Spearman ρ，并取平均作为该时间步的奖励r_t（非查询步奖励为0）。
- **策略学习**：使用PPO或其变体（VL-DAC、LOOP），仅以此奖励信号进行优化，无需任何环境提供的奖励。

### 公式（文字描述）
- 奖励函数：r_t = Spearman_ρ( 预测进度排名, 窗口内真实时间索引 ) ∈ [-1, 1]。
- 阶段1的GRPO奖励：R = Spearman_ρ( p, q )，其中p为模型输出的进度排名，q为真实时间索引。

## 7. 优点：方法与实验亮点

- **奖励设计的创新性**：使用Spearman相关系数作为奖励，有界、尺度不变，天然对校准不敏感，避免了标量奖励的过拟合和奖励破解问题。
- **两阶段解耦高效**：评分器离线训练一次后冻结，策略学习时仅需推理，计算开销可控（每N步一次VLM推理）。
- **强大的跨域泛化能力**：在视觉外观差异极大的源（GameBoy vs MetaWorld vs COIN）上训练的评分器仍能有效指导不同环境下的策略，表明序数进度信号具有跨模态通用性。
- **严谨的实验设计**：多种子、多任务、多种基线、消融充分，并额外验证了循环轨迹下的信号特性，保证了结论的可靠性。
- **可复现性**：超参数和训练细节全面公开（附录A.3-A.4）。

## 8. 不足与局限

- **依赖多样的专家数据**：如果专家视频覆盖不足，可能导致过拟合或较弱的跨级泛化（如Catrap Level 3的训练曲线显示波动，不同级别间迁移不对称）。
- **窗口大小的敏感性**：太小的窗口可能夸大局部时间一致性而忽略全局进展，太大的窗口则信号稀释；需要调参找到中间稳定区间，且不同环境最优值可能不同。
- **VLM推理延迟**：尽管采用稀疏计算（每15步一次），但仍需重型VLM，约束了实时性；同时VLM的解析失败（如输出格式错误）需特殊处理。
- **对非单调任务的支持有限**：论文虽声称不假设全局单调，但循环轨迹实验表明信号在高度非单调区域可能失效；建议结合多尺度聚合或子目标建模。
- **未涉及真实机器人或复杂长视频场景**：实验仅在模拟器（PyBoy, MuJoCo）上完成，现实世界中的视觉噪声、非静态环境等尚未验证。
- **算力成本仍较高**：尽管相对高效，但单次训练仍需多块H100 GPU，对于小型研究团队可能门槛较高。
