## 摘要
强化学习通常用于利用环境反馈训练大型语言模型。在实际应用中，环境通常提供稀疏或延迟的反馈。这使得模型难以定位其推理中的哪些行动导致了成功或失败。因此，从这些信号中有效学习是困难的，因为模型必须确定每个失败应如何为后续迭代中有意义的行为修正提供信息。我们提出了一种训练框架——自我审阅强化学习，该框架在每个强化学习回合中嵌入明确的自我审阅步骤。当首次响应失败时，模型会生成自我审阅以识别问题所在，从而为改进的第二次尝试创造条件。与推理时的反思方法（如Reflexion）不同，该框架通过策略梯度优化自我审阅，并通过选择性蒸馏将改进内化到基础策略中，确保其在未来回合中持续存在。跨回合记忆将在训练过程中遇到类似任务时，保留成功的自我审阅以便重复使用。我们在GSM8K基准测试上，使用GRPO优化器在两个语言模型Qwen 3-4B和OLMo-3-7B上评估了SRRL与标准RLVR基线的对比。SRRL在最终奖励性能上持续优于RLVR，并通过成功将反馈转化为行为改进，实现了更高的学习效率。

## Abstract
Reinforcement Learning is commonly used to train large language models using environmental feedback. In applied settings, the environment usually provides sparse or delayed feedback. This makes it difficult for the model to pinpoint which actions in its reasoning led to success or failure. So, learning effectively from these signals is hard because the model must determine how each failure should inform meaningful behavioral corrections in subsequent iterations. We introduce a training framework, Self-Review Reinforcement Learning, that embeds an explicit self-review step into each RL episode. When a first-pass response fails, the model generates a self-review to identify what went wrong, which conditions an improved second attempt. Unlike inference-time reflection approaches, such as Reflexion, the framework optimizes self-review with policy gradients and internalizes improvements into the base policy via selective distillation, ensuring they persist across future episodes. A cross-episode memory keeps successful self-reviews for reuse when encountering similar tasks in future episodes during training. We evaluate SRRL against a standard RLVR baseline using the GRPO optimizer across two language models, Qwen 3-4B and OLMo-3- 7B, on GSM8K benchmark. SRRL consistently outperforms the RLVR in final reward performance and achieves greater learning efficiency by successfully transforming feedback into behavioral improvement.

- **问题**：强化学习（RL）常用于训练大语言模型（LLM），但环境提供的奖励信号通常是稀疏、延迟的，模型难以从一次失败中定位哪些中间推理步骤导致了错误，导致学习效率低下。
- **现有方法局限**：
  - 标准RLVR（基于可验证奖励的强化学习）只给标量奖励，无结构反馈，模型需要反复探索来隐式推断修正策略。
  - 推理时反思方法（如Reflexion、Self-Refine）虽然在推理时自我修正有效，但增加了部署延迟，且改进不持久。
  - 监督微调（SFT）缺乏自我修正机制，无法利用交互反馈。
- **意义**：需要一种训练框架，能将稀疏反馈转化为显式、结构化的行为修正，并将改进持久固化到策略参数中，无需在推理时增加额外步骤。

## 2. 方法论：核心思想、关键技术细节和算法流程
### 核心思想
在标准RLVR的每个回合中嵌入一个显式的**自我审阅步骤**：当首次响应失败时，模型生成自我审阅文本（分析错误并提出修正策略），然后基于该审阅生成第二次响应。通过策略梯度优化审阅过程，并通过**选择性蒸馏**将成功修正内化到基线策略中，同时利用**跨回合记忆**保存有效审阅供后续类似任务复用。

### 关键技术细节
- **回合流程**（对应论文Figure 1）：
  1. 给定问题 \(x\)，模型 \(\pi_\theta\) 生成首次响应 \(y_1\)，环境返回标量奖励 \(r_1\) 和文本反馈 \(f_1\)。
  2. 若 \(r_1 < \tau\)（\(\tau=1\)，即失败），则触发自我审阅：模型基于 \(x, y_1, r_1, f_1\) 和检索到的记忆 \(m\) 生成审阅文本 \(\Delta\)。
  3. 模型条件于 \(x\) 和 \(\Delta\) 生成第二次响应 \(y_2\)，环境返回 \(r_2, f_2\)。
  4. 若第二次成功（\(r_2 > \tau\)），则 \(\Delta\) 存入跨回合记忆 \(m\)，并执行选择性蒸馏。
- **跨回合记忆**：只存储成功的自我审阅，作为纯文本记录，在后续训练中作为系统上下文提供给自我审阅步骤。
- **选择性策略蒸馏**：对成功的第二次响应，增加蒸馏损失：\(\mathcal{L}_{\text{distill}} = -\mathbb{E}[\mathbb{I}\{r_2 > \tau\} \log \pi_\theta(y_2 | x)]\)，训练模型直接从 \(x\) 生成 \(y_2\) 质量的响应，无需访问 \(\Delta\)。
- **RL目标**：采用GRPO（Group Relative Policy Optimization）作为基础策略优化器，策略损失为 \(\mathcal{L}_{\text{policy}} = -\mathbb{E}[A(x,y) \log \pi_\theta(y|x)]\)，其中优势 \(A\) 通过组内归一化计算。自我审阅 \(\Delta\) 的奖励设为 \(r_2\)。
- **组合目标**：\(\mathcal{L}_{\text{SRRL}} = \mathcal{L}_{\text{policy}} + \mathcal{L}_{\text{distill}} + \beta \cdot \mathcal{L}_{\text{KL}}\)，KL系数 \(\beta=0.001\)。
- **关键设计原则**：自我审阅仅在首次响应失败时触发（防止奖励破解和策略漂移）；最终策略在部署时不需任何审阅、记忆或蒸馏，零额外延迟。
