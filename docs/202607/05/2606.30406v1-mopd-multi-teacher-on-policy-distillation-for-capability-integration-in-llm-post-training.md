## 摘要
现代大语言模型（LLM）在后训练阶段依赖强化学习来提升特定能力，但将多种能力集成到单一模型中仍然困难。现有的方法，如离线策略微调（Off-Policy Finetune）和混合强化学习（Mix-RL），要么效率低下，要么性能损失。本文提出了一种多教师在线策略蒸馏（MOPD）范式，用于组合多个领域强化学习教师的能力：我们首先进行按领域的专门强化学习，得到一组领域教师，然后在其自身的轨迹上将这些教师蒸馏给学生模型。这消除了暴露偏差并提供了密集的优化信号。在Qwen3-30B-A3B上，MOPD优于Mix-RL、级联强化学习（Cascade RL）、Off-Policy Finetune和参数合并（Param-Merge）基线，几乎继承了每位教师的全部能力。MOPD还支持领域教师的并行独立开发，消除了多领域后训练中典型的跨领域耦合。MOPD已部署在工业级前沿模型MiMo-V2-Flash的后训练中，展示了其在前沿规模LLM能力集成中的实际价值。

## Abstract
Modern large language models (LLMs) rely on reinforcement learning during post-training to push specific capabilities, yet integrating multiple capabilities into one model remains hard. Existing methods, such as Off-Policy Finetune and Mix-RL, are either inefficient or lose performance. In this work, we propose Multi-teacher On-Policy Distillation (MOPD), a post-training paradigm for combining the capabilities of multiple domain RL teachers: we first run per-domain specialised RL to obtain a set of domain teachers, then distill these teachers into the student on its own rollouts. This eliminates exposure bias and provides a dense optimization signal. On Qwen3-30B-A3B, MOPD outperforms Mix-RL, Cascade RL, Off-Policy Finetune, and Param-Merge baselines, inheriting nearly all of each teacher's capability. MOPD also enables parallel, independent development of domain teachers, removing the cross-domain coupling typical of multi-domain post-training. MOPD has been deployed in the post-training of MiMo-V2-Flash, an industrial-scale frontier model, demonstrating its practical value for capability integration in frontier-scale LLMs.

### 论文核心问题与整体含义（研究动机和背景）
大型语言模型（LLM）在后训练阶段通常依赖强化学习（RL）来提升特定领域的能力（如数学推理、指令跟随、软件工程等）。然而，将多种能力集成到单一模型中仍是一个开放难题。现有方法主要分为四类：
- **混合RL（Mix-RL）**：将所有领域的提示混合进行联合RL训练，但不同领域的训练信号相互干扰，产生“跷跷板效应”，集成模型性能低于各领域教师。
- **级联RL（Cascade RL）**：按顺序训练各个领域，但早期阶段的能力会在后续训练中衰退，且整个训练过程稳定性风险高。
- **离线策略微调（Off-Policy Finetune）**：先在每个领域训练RL教师，然后用教师的输出对学生进行SFT微调，但这属于离线策略，存在经典的**暴露偏差**（训练与推理分布不匹配）。
- **参数合并（Param-Merge）**：在权重空间对教师参数进行平均或任务向量合成，但融合模型通常不稳定，难以完整继承所有教师的能力。

因此，论文旨在提出一种新的后训练范式，既能高效集成多个领域教师的能力，又能消除暴露偏差，同时支持并行开发。

### 论文提出的方法论：核心思想、关键技术细节
**核心思想**：MOPD（Multi-teacher On-Policy Distillation）是一种三阶段后训练范式，先在各个领域独立训练专用RL教师，然后通过**在线策略蒸馏**将多个教师的能力整合到学生模型中。具体流程：
1. **第一阶段：通用SFT**。在一个覆盖所有目标领域的通用数据集上微调基座模型，得到共享的SFT检查点。
2. **第二阶段：领域专门RL训练**。以第一阶段SFT检查点为初始，为每个领域独立训练一个专门的RL教师（例如数学用可验证答案RL，软件工程用可执行沙盒agent RL）。各领域教师训练可完全并行。
3. **第三阶段：MOPD蒸馏**。学生模型初始化为第一阶段的SFT检查点，冻结第二阶段的多位教师。训练时：①从数据集采样一批提示；②学生为每个提示生成一条轨迹（rollout）并记录每步token概率分布；③将每条轨迹根据领域路由到对应的教师，教师对该轨迹进行prefill（计算自身在该轨迹上的逐token log概率）；④学生通过最小化自身与教师之间的**逐token反向KL散度**进行更新。

**关键技术细节**：
- **暴露偏差消除**：训练完全基于学生自己的rollout分布，训练与推理分布一致。
- **密集逐token监督**：教师提供每个token位置的概率分布，比轨迹级奖励信号更密集，降低方差、提高样本效率。
- **策略空间集成**：通过提示路由将不同教师的能力在策略空间融合，避免权重空间融合的不稳定性。
- **两种损失实现**：
  - **策略梯度形式**：将KL散度的梯度转化为策略梯度，以教师与学生log概率差作为每一步的advantage，并使用双向裁剪。公式：\( \hat{A}_{\text{MOPD},t} = \text{sg}[\log \pi_{\phi_d}(y_t) - \log \pi_{\theta}(y_t)] \)，损失为 \( -\mathbb{E}[\frac{1}{|y|}\sum_t \hat{A}^{\text{clip}}_{\text{MOPD},t} \log \pi_{\theta}(y_t)] \)。
  - **Top-k蒸馏形式**：只对教师分布的top-k token进行反向KL蒸馏，并加入校正项避免截断偏差，降低通信和计算开销。公式中加入了 \( \pi_{\phi_d}(v) - \pi_{\theta}(v) \) 项确保满秩最优解。
- **基础设施**：教师以独立Prefill服务形式运行，与学生采样异步并行，实际几乎无墙钟开销。

### 论文的主要结论与发现
1. **MOPD在Qwen3-30B-A3B上取得最高归一化得分0.937**，优于最强基线Mix-RL（0.882），提升5.5个点。
2. **MOPD几乎完整继承了每个教师的能力**：每个领域归一化得分在0.91~0.95之间，范围仅0.044，远小于其他方法（Cascade RL范围0.41，Off-Policy Finetune范围0.36）。
3. **MOPD样本效率显著更高**：在指令跟随和软件工程上约25K~30K样本即达到教师水平，而Mix-RL需要150K~180K。
4. **同源教师至关重要**：若使用分布差异大的外部教师（Qwen3-235B-A22B），训练不稳定甚至崩溃，KL初始值高5倍，熵收缩。
5. **多轮迭代可进一步提升**：第二轮MOPD使归一化得分从0.937提升到0.986，且第二轮教师更优（归一化1.030）。
6. **工业规模验证有效**：在MiMo-V2-Flash上，MOPD在大部分基准上匹配或超过对应教师，仅IFBench和SWE-Bench有微小下降。
