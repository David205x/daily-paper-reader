## 摘要
近期在大规模序列建模方面的进展表明，单个模型能够从高度多样化的数据分布中学习有用的表示。受这些进展的启发，我们研究是否可以在大量异构强化学习环境的集合上训练统一的Transformer策略。我们引入了LDM-v0，这是一个大型决策模型，在来自数千个跨越多个领域和模态的环境的轨迹上离线训练。LDM-v0是一个多任务、多模态的Transformer策略，以观测、动作、奖励和终止信号的历史为条件，并通过监督式的下一动作预测在离线轨迹上进行训练。我们描述了用于构建LDM-v0的环境基础设施、自动化数据生成流水线、模型架构和训练方法，并评估了其在多样化环境中的性能。我们表明，单个预训练模型在大约1000个环境（包括机器人、自动驾驶、库存管理、网络安全、交易和视频游戏）上匹配了独立训练的特定任务参考策略的性能。这些结果证明了使用单个Transformer策略在异构强化学习环境中进行大规模离线预训练的可行性。

## Abstract
Recent progress in large-scale sequence modeling has shown that a single model can learn useful representations across highly diverse data distributions. Inspired by these advances, we investigate whether a unified transformer policy can be trained across large collections of heterogeneous reinforcement learning environments.   We introduce LDM-v0, a Large Decision Model trained offline on trajectories collected from thousands of environments spanning multiple domains and modalities. LDM-v0 is a multi-task, multi-modal transformer policy conditioned on histories of observations, actions, rewards, and termination signals, and trained through supervised next-action prediction over offline trajectories. We describe the environment infrastructure, automated data generation pipeline, model architecture, and training methodology used to build LDM-v0, and evaluate its performance across diverse environments. We show that a single pretrained model matches the performance of independently trained task-specific reference policies on approximately 1,000 environments including robotics, autonomous driving, inventory management, cybersecurity, trading, and video games. These results demonstrate the feasibility of large-scale offline pretraining across heterogeneous reinforcement learning environments using a single transformer policy.

## 2. 论文方法论

### 核心思想
- 将多任务 RL 转换为序列建模问题：训练一个自回归 transformer，以历史轨迹（观测、动作、奖励、终止信号）和当前观测为条件，预测下一动作（监督学习）。
- 利用自动化流水线生成高质量离线数据：先训练大量任务特定参考策略，再用最终策略回放标注轨迹，然后训练 LDM-v0 模仿这些参考动作。

### 关键技术细节
- **自动参考策略生成**：

  - 过滤：剔除无显著学习效果或最终性能低于最佳策略 95% 的轨迹。
- **统一 tokenization**：

- **主要贡献**：证明了跨异构 RL 环境的大规模离线预训练是可行的，并提供了完整的基础设施、数据生成流程和模型设计。

## 7. 优点

- **大规模自动化基础设施**：通过 Docker 容器化和自动化流水线集成了 146 个环境库，解决了依赖冲突问题；数据生成采用 AutoRL 思路自动选择参考策略，减少人工干预。
- **紧凑的 tokenization 设计**：将每个 transition 压缩为一个元 token（而非每个维度单独 tokenize），显著降低序列长度（对比 Gato 每个 timestep 需上百 token），使得在 2048 上下文内可容纳更多历史步。
- **统一多模态架构**：支持连续、离散、图像等多种观测和动作空间，且无需手工特征工程。
- **实验结果宽广**：在数千个真实世界应用环境上验证，具有较高的工程价值和实用性。

## 8. 不足与局限

- **泛化能力未评估**：论文明确声明主要目标是“是否能在训练分布内联合建模多种行为”，而非 out‑of‑distribution 泛化；未来需研究零样本适应、微调等能力。
- **数据覆盖有限**：仅使用了约 19% 的环境库（实际训练约 3000 个，总池约 15000 个），导致模型可能未充分学习某些领域模式。
- **消融实验不足**：缺少对骨干架构、tokenization 方式、数据规模、参考策略选择策略等的系统性消融分析。
- **缺少直接对比**：未与 Gato、JAT 等现有通用 agent 在同一环境集合上定量比较，难以判断自身方法相对增益。
- **计算开销极大**：数据生成 12 周、29TB 存储、16 块 H200 训练 6 天，可复现性和实用性受限。
- **潜在的偏差风险**：参考策略仅基于 Stable‑Baselines3 和 SB3‑Contrib 中的有限算法，且只保留 top‑3 配置，可能漏掉对某些环境更优的方法；数据过滤可能剔除有探索价值的轨迹。
