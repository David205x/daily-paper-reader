## 摘要
在知识密集型问答中，LLM智能体在对其当前答案是否不确定、无依据或已完成缺乏完整了解的情况下执行检索和推理行动。这导致两种失败模式：一是给出自信但无依据的答案，损害准确性；二是当已有证据足以回答时过度检索，造成计算浪费。为了让智能体更全面地了解其操作的状态空间，我们引入了校准验证器遥测（CalVerT），它通过额外的遥测数据（校准的自信心分数和基于事实的验证器分数）来增强智能体的状态。我们展示了CalVerT在无训练和基于训练的设置中都能改进智能体。在四个问答基准上，我们发现CalVerT通过在智能体过度依赖参数知识的情况下触发检索来提高F1值，同时在智能体拥有足够上下文作答时减少冗余检索。我们证明CalVerT可以在不训练的情况下增强现有问答框架。此外，CalVerT也改进了训练系统：通过简单地为智能体状态添加遥测数据，我们在强化学习后观察到相比具有相同训练但无CalVerT遥测的智能体有所改进。

## Abstract
LLM agents in knowledge intensive question answering take retrieval and reasoning actions with incomplete knowledge about whether their current answer is uncertain, unsupported, or already complete. This produces two failure modes: committing to confident but unsupported answers, which hurts accuracy, and over-retrieving when the evidence in hand already suffices, resulting in wasted compute. To give agents a more complete picture of the state space they are operating in, we introduce calibrated verifier telemetry (CalVerT), which augments the agent's state with additional telemetry: a calibrated self-confidence score and a grounding verifier score. We show that CalVerT can improve agents in both training-free and training-based settings. On four QA benchmarks, we find that CalVerT raises F1 by triggering retrieval in cases where agents over-rely on parametric knowledge, while cutting redundant retrieval in cases where agents have sufficient context to answer. We show that CalVerT can augment existing QA frameworks without training. Moreover, CalVerT also improves trained systems: by simply augmenting an agent's state with telemetry, we observe improvements after reinforcement learning, as compared to an agent with identical training but no CalVerT telemetry.

## 1. 核心问题与整体含义（研究动机和背景）

**问题**：LLM 智能体在知识密集型问答（如多跳 QA）中，由于无法感知当前答案是否不确定、是否缺乏证据支持或是否已经完整，导致两大失败模式：
- **过度信任参数知识**：智能体过早承诺一个自信但无依据的错误答案，损害准确性；
- **过度检索**：即便已有足够证据，智能体仍继续检索，浪费计算资源。

**动机**：现有智能体通常只依赖单一信号（如自信度或证据支持度）来决定下一步行动，无法有效平衡准确性与效率。需要一种能同时提供两种正交信号的测量机制，帮助智能体做出更优决策。

**整体含义**：通过为智能体状态添加校准的自信度与证据支持度遥测信息，可以显著提升其在知识密集型任务中的行动质量与学习效率，且方法轻量易移植。

## 2. 方法论

**核心思想**：在每一轮智能体决策前，暴露两种遥测信号：
- **校准的自信度分数（Calibrated Confidence）**：来自 DiNCo（Wang & Stengel-Eskin, 2026），包含归一化语言自信度（NVC）和自一致性率（SC），综合为 `final_conf = 0.5*(NVC + SC)`。
- **证据支持度分数（Grounding Score）**：由 MiniCheck 验证器（Bespoke-MiniCheck-7B）对候选答案的每个声明进行证据蕴含评估，取均值 `g_mean` 和最小值 `g_min`。

**关键技术细节**：
- 智能体采用 ReAct 风格循环，动作集：`{commit, retrieve, refine, decompose}`。
- 遥测信号不经阈值硬编码，而是直接以自然语言形式加入提示词（prompt），让智能体自主推理。
- 在**无训练**场景：直接将遥测信号插入冻结智能体的提示中，引导其决策。
- 在**训练**场景：使用 GRPO（Group Relative Policy Optimization）对智能体策略进行强化学习，将遥测信号作为状态的一部分。

**算法流程**（文字说明）：
1. 规划器将问题分解为子目标 DAG。
2. 每一轮智能体观察当前子问题、证据池；计算当前候选答案的自信度（DiNCo）和证据支持度（MiniCheck）。
3. 智能体根据提示（含遥测信号）选择行动：`commit`（提交答案）、`retrieve`（检索更多证据）、`refine`（基于现有证据修改答案）、`decompose`（分解子问题）。
4. 循环直到子目标全部解决或耗尽预算，最后由合成器生成最终答案。

## 6. 主要结论与发现

1. **遥测信号提升准确性和效率**：在多数设置下，F1提升（最多+15.4%），同时减少冗余动作（多跳任务中动作轮次下降，单跳任务中适度增加检索）。
2. **框架无关性**：遥测信号可轻松替换五个现有框架的检索门，提升性能或减少检索次数。
3. **训练增益**：GRPO训练中，添加遥测的智能体显著优于无遥测的相同训练过程（F1 +5.9% on 8B，+3.3% on 30B）。
4. **双信号必要性**：单独自信度或单独证据支持度均不如两者结合（多跳场景尤其明显）。
5. **难度适应**：遥测信号对越难的问题（多跳数高、实体罕见）增益越大，能自动将计算预算导向困难样本。
6. **模型规模要求**：无训练设置中，小模型（8B）无法有效利用遥测信号，但训练可弥补。

## 7. 优点

- **方法简单高效**：仅通过向提示添加遥测信号（无阈值、无重训练）即可提升性能，兼容多种现有框架。
- **双信号正交互补**：自信度追踪“是否应提交”，证据支持度追踪“提交是否安全”，覆盖了智能体决策的两个关键维度。
- **训练与无训练双模式支持**：提供了灵活的应用方案，尤其无训练方案可即插即用。
- **校准严格**：对自信度和证据支持度均进行了校准评估（ECE、Brier、AUROC），确保信号可靠。
- **详细消融与分析**：深入分析了不同难度、不同动作选择、不同信号组合的影响，结论扎实。

- **仅适用于单答案多跳QA**：论文指出CalVerT的提交聚焦于单一最终答案，不适用于需要列举所有相关项的任务（如AmbigQA、FanOutQA）。这是一个任务假设上的局限，而非方法的根本缺陷。
- **小模型在无训练设置下效果差**：Qwen3-8B在无训练时表现不如基线，说明遥测信号需足够大的模型才能自主解释利用。好在此问题可通过训练缓解。
