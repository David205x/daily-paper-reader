## 摘要
具有可验证奖励的强化学习（RLVR）提升了语言模型的推理能力，但GRPO风格的优化仍容易崩溃。我们通过令牌级梯度动力学分析了这种不稳定性，推导出一个分类体系，用于预测更新如何影响下一令牌概率和熵。该分类表明，稳定性共同取决于当前策略下的优势符号和令牌分布。受此发现启发，我们提出胜者优势策略优化（WAPO），一个简单的在线裁剪策略梯度目标，仅对正优势补全进行更新。在数学推理和多跳问答基准测试中，WAPO提升了训练稳定性，并在多个模型系列中达到或超越基线表现。完整代码可在 https://github.com/layer6ai-labs/wapo 获取。

## Abstract
Reinforcement learning with verifiable rewards (RLVR) improves language-model reasoning, but GRPO-style optimization remains prone to collapse. We analyse this instability through token-level gradient dynamics, deriving a taxonomy that predicts how updates affect next-token probabilities and entropy. The taxonomy shows that stability depends jointly on the advantage sign and token distribution under the current policy. Motivated by this finding, we propose Winner Advantage Policy Optimization (WAPO), a simple online clipped policy-gradient objective that updates only on positive-advantage completions. Across mathematical reasoning and multi-hop QA benchmarks, WAPO improves training stability and matches or outperforms baselines across multiple model families. Full code can be found at https://github.com/layer6ai-labs/wapo.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：RLVR（如GRPO）通过离线采样更新策略，能有效提升语言模型的推理能力，但训练过程极易发生两种类型的崩塌（collapse）：**高熵崩塌**（生成无意义、多语言乱码）和**低熵崩塌**（生成重复、简短的模板答案）。
- **现有思路的局限**：现有工作通常将崩塌归因于旧策略与新策略之间的分布偏移，并试图通过裁剪重要性比率、信任区域约束等方式来缓解，但实验发现**即使更激进地裁剪发散性token也会导致崩塌**，说明单一的距离约束无法解释稳定性。
- **本文视角**：转而分析**梯度更新在当前策略下对局部 token 概率分布和熵的一阶影响**，从而揭示稳定性本质上依赖于**优势符号（advantage sign）** 和**当前 token 在分布中的位置（峰/谷）**。这一新视角为设计更鲁棒的RLVR算法奠定了基础。

## 2. 论文提出的方法论

- **核心思想**：通过分析 token 级别梯度的局部效应，发现更新对不同 token 概率和熵的影响有四种模式（正-峰、正-谷、负-峰、负-谷），其中正-峰稳定但保守，正-谷和负-峰会增加熵导致高熵崩塌，负-谷降低熵导致低熵崩塌。
- **关键分类（peak / valley）**：定义参考水平 \( C(p)=\sum p_j^2 \)，若采样 token 的概率 \( p_s \geq C(p) \) 则称其为 `peak`，反之为 `valley`。peak 多为高概率 token，valley 多为低概率 token。
- **梯度公式推导**（公式1、3）：
  - 非采样 token i 的概率变化：\(\Delta p_i = \eta A p_i (C(p) - p_s - p_i) + O(\eta^2)\)
  - 熵变化：\(\Delta H = -\eta A [p_s \log p_s - \sum p_j^2 \log p_j + H(p)(p_s - C(p))] + O(\eta^2)\)
  - 符号分析表明：Pos-valley 和 Neg-peak 增加熵，Pos-peak 和 Neg-valley 减少熵。
- **WAPO 算法**：
  - 仅保留 **优势为正** 的完成序列（即 \( A_i > 0 \) 的样本），进行在线裁剪策略梯度更新。公式（8）：
    \[
    J_{\text{WAPO}}(\theta) = \mathbb{E}_{x \sim \mathcal{D}}\left[ \frac{1}{G T} \sum_{i=1}^{G} \sum_{j=1}^{T_i} A_i^+ \cdot \min(\rho_{ij}(\theta), 1+\epsilon) \right], \quad \text{only if } \exists i: A_i^+ > 0
    \]
  - 使用组归一化优势、最大化序列长度 T 统一归一化，避免长度偏差。
  - 在二元奖励下，梯度方向与成功概率 \( q_x \) 的梯度一致，但带有自适应因子 \( 1 - q_x \)，使得容易 prompt 的更新幅度自然减小，困难 prompt 获得更大更新。

## 6. 论文的主要结论与发现

- WAPO **显著提升训练稳定性**，在所有实验设置中均未发生崩塌，而 DAPO 在 6 个多跳 QA 中有 4 个设置崩塌。
- 在多跳 QA（较难任务）上，WAPO **性能大幅领先**稳定基线（如 OTT-QA Qwen: +9.9% EM, Hotpot-QA Gemma: +10.6%）。
- 在数学推理上，WAPO **与最强基线（GRPO、GSPO）相当**，早期略低但后期持平。
- WAPO 的 pass@k 表现同样优秀，证明其**保留了探索多样性**，未因只使用正样本而过度收敛。
- 域外迁移（Hotpot→2wiki）WAPO 均为最优，数学迁移则与其他方法持平。
- 梯度分类预测（peak/valley）在 Qwen、Smol、Gemma 模型上均得到验证，**理论指导性强**。

## 7. 优点

1. **理论深度**：从一阶梯度推导出熵及概率变化的明确方向，提出可验证的 peak/valley 分类，对理解 RLVR 崩塌提供了根本性机制。
2. **方法简洁有效**：WAPO 仅通过 masking 操作移除负优势更新，不增加额外超参数或信任区域约束，易于实现且效果显著。
3. **实验全面**：覆盖数学推理和多跳问答两大场景、三种不同模型系列、域内外评估，充分证明方法的鲁棒性和泛化性。
4. **与现有方法正交**：WAPO 可以与更好的 rollout 机制、KL 正则等结合，具有扩展潜力。

## 8. 不足与局限

1. **实验范围有限**：仅测试了 3-4B 参数的中小规模模型（LoRA rank 8），未在更大模型（如 7B、13B）或 MoE 架构上验证，也未使用全参数微调（full FT）。
2. **数学任务上的早期性能损失**：WAPO 在数学推理任务初期略低于基线，可能源于自适应因子 \(1-q_x\) 在低成功率时更新幅度较小，启发式地丢失了部分早期探索信号。
3. **对二进制奖励的依赖**：理论分析假设奖励为 0/1 二元，虽通过组正常化适应连续奖励，但未考虑更复杂的奖励函数（如过程奖励、惩罚性奖励）。
4. **统计显著性缺失**：未报告多次运行的均值和方差，无法判断性能差异的显著程度。
5. **可能丢弃有效学习信号**：负优势的低概率 token 在某些场景下（如纠正错误思路）可能提供有用梯度，但正-only 策略彻底丢弃它们，论文也承认这是简化假设。
