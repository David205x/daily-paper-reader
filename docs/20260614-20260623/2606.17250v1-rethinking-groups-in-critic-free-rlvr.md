## 摘要
强化学习已成为大型语言模型后训练的核心范式。现有的无评论家RL方法通常为同一个问题生成一组展开轨迹，以估计用于优势计算的价值基线。然而，这种设计存在数据效率低、组同步障碍以及与结构化展开不灵活等问题。在这项工作中，我们重新审视了“组”的作用，并表明其根本功能并非仅仅是估计基线，而是防止对负样本的错误惩罚。基于这一见解，我们提出了负标记过滤策略，这是一种简单有效的方法，能够实现稳定的单展开轨迹训练。我们将其应用于两种批次级优势方法，在推理任务上取得了与基于组的RL技术相当的性能，在智能体任务上则表现更强。

## Abstract
Reinforcement learning (RL) has become a central paradigm for post-training large language models. Existing critic-free RL methods typically generate a group of rollouts for the same question to estimate value baselines for advantage computation. However, this design suffers from data inefficiency, group synchronization barriers, and inflexibility with structured rollouts. In this work, we revisit the role of the ``group'' and show that its underlying function is not merely to estimate baselines but to prevent false penalties on negative samples. Building on this insight, we propose negative token filtering, a simple and effective strategy that enables stable single-rollout training. We apply it to two batch-level advantage methods, achieving comparable performance on reasoning tasks and stronger performance on agentic tasks relative to group-based RL techniques.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：现有无评论家强化学习（Critic-Free RLVR）方法（如 GRPO、RLOO、ReMax）依赖于为每个提示生成多个 rollout 组成“组”，并利用组内统计量估计价值基线。这种设计导致数据效率低下（需丢弃组内奖励相同的 rollout）、同步障碍（多 rollout 需等待所有生成完成），且难以处理结构化 rollout（如智能体任务中的多步交互）。
- **核心洞察**：论文重新审视“组”的作用，发现其根本功能并非单纯的基线估计，而是**防止对负样本中的支持性 token 的错误惩罚**。支持性 token 包括格式、中间推理步骤、工具调用等共享的、有用的语言模式，这些在正负轨迹中均出现。若在单 rollout 训练中对所有 token 一视同仁地惩罚，会导致模型快速遗忘核心能力（灾难性遗忘）。分组通过正负轨迹的梯度抵消保护这些 token。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：负轨迹并非完全错误，其中的高概率 token 往往是支持性 token。通过在负轨迹中只惩罚低概率 token（最可能导致失败的 token），可以避免有害更新，实现稳定的单 rollout 训练。
- **关键技术细节**：
  - **负令牌过滤（Negative Token Filtering, NTF）**：对于每个负轨迹，按当前策略下各 token 的概率进行排序，设定掩蔽分数 τ（默认 0.1），屏蔽概率最高的前 τ 比例的 token。保留低概率的 token 集合 K(o) 参与梯度计算，分母仍使用序列全长归一化，使被屏蔽的 token 对梯度无贡献。公式如下：
    \[
    \ell^-(o) = \frac{1}{|o|} \sum_{t \in K(o)} \max\left(\rho(o_t), 1-\epsilon\right)
    \]
  - **应用方法**：
    - **REINFORCE++ (RF++)**：批次级 z-score 归一化优势。
    - **Contrastive-REINFORCE (C-RF)**：去除式(6)中加权系数的对比 REINFORCE，避免在初始成功率极低时信号衰减。
  - **理论支撑**：
    - 统计分析了正负轨迹的 token 重叠（图4），发现高概率 n-gram 在正轨迹中有更高的命中率。

## 6. 论文的主要结论与发现

1. **组的作用是保护共享支持性 token，而非单纯估计基线**。单 rollout 训练的不稳定源于负样本对高概率支持性 token 的过度惩罚。
2. **NTF 可使单 rollout 训练稳定**：通过屏蔽负轨迹中前10%的最高概率 token，C-RF w/ NTF 在1.5B模型上平均准确率仅比 GRPO（G=16）低0.91点，且显著优于 G=2 的 GRPO。
3. **在智能体任务上，C-RF w/ NTF 超越所有基线**：ALFWorld 上1.5B模型达87.38% in-distribution success，7B达90.48%，均超过 PPO、RLOO、GRPO。
4. **分组梯度抵消机制**：通过 token 重叠统计和梯度子空间投影验证，正负轨迹的高概率 token 梯度高度对齐且符号相反，分组可有效抵消有害更新。
